DAoE: Dark Age of Exiles
========
based on Dawn of Light

City of Heroes meets Path of Exiles
----
Why not also throw in some Dark Souls while we are at it? So many great games, let's satirize them all ;)

I'm using DOL as the server implementation of a more exciting game.

To do this, I'm building an extremely customised DOL core, to be used more as a general purpose MMO engine. It compiles so far, but is untested.
